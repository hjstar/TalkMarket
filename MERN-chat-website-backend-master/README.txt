To run this backend server at your pc follow the given steps -:

Install NodeJS in your PC if it's not already installed.
Clone the repository using "git clone https://github.com/Prateek88786/MERN-chat-website-backend.git" command in the terminal.
Change the directory using "cd MERN-chat-website-backend/" command.
Run the "npm i" command to install the dependencies.
Now run "npm start" to start the server.
Also run the Frontend server using steps given in frontend repository whose link is mentioned in the description.
